Windows-RasPhone
================
###RasPhone and RasDial
    
    Rasphone 
      ---------------------------
      拨号网络命令行
      ---------------------------
      Rasphone [-f file] [[-e | -d | -h | -r] entry]
      
      Rasphone [-f file] -a [entry]
      
      Rasphone [-f file] -lx link
      
      Rasphone 
      
      
      
      -a	弹出新建项目对话框
      -e	弹出编辑项目对话框
      -d	弹出拨号项目对话框
      -h	不知不觉地挂断项目
      -r	不知不觉地删除项目
      -lx	在拨号快捷方式文件上执行命令 'x'
      x	任何命令 a、e、d、h 或 r
      entry	操作应用的项目名
      file	电话簿文件的完整路径
      link	拨号快捷方式文件的完整路径
      
      'entry' 单独选择电话簿对话中的项目
